## 1. 产品概述

面向科普馆的"小行星撞击与行星防御"交互推演工具，让观众通过设定参数直观理解近地天体撞击地球的威力，以及动能撞击器偏转小行星轨道的原理。纯前端运行，所有物理计算在浏览器端完成，使用公认经验公式确保量级正确。

- 目标用户：科普馆参观者（青少年及成人），无需专业背景
- 核心价值：将抽象的天文数字转化为可感知的灾难规模，并传达"越早干预、所需力量越小"的行星防御核心理念

## 2. 核心功能

### 2.1 用户角色
无需注册登录，所有访客均可直接使用。

### 2.2 功能模块
1. **撞击效应推演页**：参数输入 → 实时计算 → 影响范围可视化 → 灾害等级判定
2. **轨道偏转推演页**：威胁设定 → 动能撞击器参数 → 偏转效果推演 → 干预时机对比

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 撞击效应推演 | 参数面板 | 输入小行星直径（10m-500km）、密度类型（石质/铁质/彗星）、撞击速度（11-72 km/s）、撞击角度（10°-90°）；提供预设场景快捷按钮（通古斯、希克苏鲁伯、恐龙灭绝等） |
| 撞击效应推演 | 结果数据卡片 | 显示撞击动能（TNT当量+广岛原子弹倍数）、撞击坑直径、地震震级、热辐射半径及伤害等级、冲击波半径及伤害等级、整体灾害等级 |
| 撞击效应推演 | 影响范围SVG示意图 | 同心圆分层展示不同距离的影响范围（热辐射区、冲击波破坏区、地震有感区等），标注关键距离数值 |
| 轨道偏转推演 | 威胁设定面板 | 设定小行星参数、距撞击剩余时间（1天-10年）、预估撞击点 |
| 轨道偏转推演 | 偏转参数面板 | 设定动能撞击器赋予的 delta-v（0.001-10 cm/s）；实时计算偏移距离 |
| 轨道偏转推演 | 轨道示意SVG | 简化2D轨道图，展示干预前后小行星轨迹差异，标注地球、偏移距离、安全边界 |
| 轨道偏转推演 | 时机对比面板 | 图表展示"相同偏移距离下，剩余时间与所需 delta-v 的关系"——体现越早干预 delta-v 越小的核心科普点 |

## 3. 核心流程

### 3.1 撞击效应推演流程
用户打开页面 → 看到默认参数（如通古斯事件预设）→ 调整参数滑块/选择密度类型 → 实时计算并更新结果 → 查看数据卡片和影响范围图 → 切换预设场景对比不同规模撞击

### 3.2 轨道偏转推演流程
用户打开页面 → 设定威胁小行星参数 → 设定距撞击剩余时间 → 调整 delta-v → 观察偏移距离是否足够脱靶 → 调整剩余时间观察所需 delta-v 变化 → 理解"越早干预越容易"的核心概念

## 4. 用户界面设计

### 4.1 设计风格
- **主题色调**：深空黑底（#0a0e1a）+ 星云蓝紫渐变点缀 + 警示橙红高亮，营造宇宙深邃感与危机紧迫感
- **辅助色**：数据绿（安全/通过）、警示黄（警告）、危险红（灾难），用于灾害等级和脱靶判定
- **字体**：标题使用 Orbitron（科幻风格显示字体），正文使用 Noto Sans SC（中文清晰可读）
- **布局**：左右分栏布局，左侧参数面板、右侧结果展示；移动端上下堆叠
- **按钮风格**：圆角胶囊形，带发光边缘效果，符合科幻主题
- **图标风格**：线性描边图标，宇宙/科技主题

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 首页/导航 | 模式选择 | 深空背景 + 两个大型选择卡片（撞击/偏转），悬停发光效果 |
| 撞击效应推演 | 参数面板 | 滑块（直径/速度/角度）+ 密度类型选择器（三个图标按钮）+ 预设场景快捷按钮 |
| 撞击效应推演 | 结果数据卡片 | 网格排列的数据卡片，每张卡片带图标+数值+单位+参考对比 |
| 撞击效应推演 | 影响范围SVG | 同心圆分层，从内到外：火球区(红)→热辐射区(橙)→冲击波破坏区(黄)→有感区(绿)，中心标注撞击点 |
| 轨道偏转推演 | 威胁面板 | 紧凑参数输入区 + 简要威胁描述 |
| 轨道偏转推演 | 轨道示意SVG | 2D俯视图，地球居中，两条轨迹线（原轨迹虚线+偏转后实线），偏移距离标注 |
| 轨道偏转推演 | 时机对比图表 | 曲线图：X轴=剩余时间，Y轴=所需delta-v，标注"安全脱靶线" |

### 4.3 响应式设计
- 桌面优先设计（科普馆大屏/触控一体机为主要场景）
- 最小支持 1024px 宽度
- 移动端自适应：参数面板和结果区域上下堆叠，SVG图表缩放

### 4.4 动效设计
- 页面切换：星空粒子流动过渡
- 参数变化时：数据卡片数值动画过渡（数字滚动效果）
- 影响范围图：同心圆扩展动画
- 轨道图：偏转轨迹动态绘制
- 脱靶判定：安全/危险状态切换时的颜色脉冲动画
